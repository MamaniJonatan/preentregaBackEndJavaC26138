# Sistema de Gestión de Productos y Pedidos (TechLab)

Este es un sistema de consola desarrollado en Java puro para la preentrega del curso.

## 🚀 Cómo ejecutar el proyecto
1. Abra el proyecto en IntelliJ IDEA o VS Code.
2. Diríjase a `src/com/techlab/Main.java`.
3. Haga clic en **Run**.

## 📸 Ejemplo de funcionamiento en consola:
```text
=== MENÚ PRINCIPAL ===
1) Agregar producto
2) Listar productos
3) Buscar/Actualizar producto
...
Elija una opción: 2
